Devices used in Networking

Router

A router is a device that forwards data packets between computer networks. Routers operate at the network layer, which is Layer 3 of the OSI Model.

Learn More
Switch

A switch is a device that connects devices together on a computer network by using packet switching to forward the packets to their destinations. Switches operate at the data link layer, which is Layer 2 of the OSI Model.

Learn More
Firewall

A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules. Firewalls operate at the application layer, which is Layer 7 of the OSI Model.
